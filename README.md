## Quick Start

```bash
# Install Dependencies
composer install

# Create Database(Tamirwellness) before Migrate

# Run Migrations
php artisan migrate

# Import Articles
php artisan db:seed

# If you get an error about an encryption key
php artisan key:generate
```
